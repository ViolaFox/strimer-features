How to use with OBS:
1. Adjust all settings on this page
2. Click Preview to test the effect
3. Click Copy URL — the URL encodes all your settings
4. In OBS: add Browser Source → paste the URL into the "URL" field
5. Set width/height to your stream resolution (e.g. 1920x1080)
6. To update: change settings → copy new URL → in OBS right-click the source → Properties → paste new URL → OK

GitHub Pages: Push this file to a repo, enable Pages, then use the GitHub URL. Font scanning works even better over HTTPS.

Why no live sync? OBS uses its own Chromium, separate from your browser. BroadcastChannel, localStorage events, and SharedWorker cannot cross browser instances. The URL method is the only serverless option.
